# CodeAlpha Student Grade Tracker

## Description
This is a Java console-based application developed as part of the CodeAlpha internship.
It allows users to add student details, calculate grades, and generate reports.

## Features
- Add student details
- Calculate total and average marks
- Assign grades
- Display student report

## Technologies Used
- Java
- Command Line Interface (CLI)

## How to Run
1. Open Command Prompt
2. Navigate to the project folder
3. Compile the files:
   javac Student.java GradeTracker.java
4. Run the program:
   java GradeTracker
